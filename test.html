Les bases

On utilise le tag <!DOCTYPE html> au début du document. Ensuite vient le tag <html> qui entoure le document Puis on divise le document en deux parties:
- <head> on y écrit les métadonnées
- <body> on y écrit la page
Les éléments basique sont les paragraphes <p>, le lien <a> et l'image <img>.

Les attributs

Pour indiquer la langue <html lang="fr>.
Pour écrire la tooltip d'un paragraphe <p title="I'm a tooltip">.
Pour écire un lien <a href="https://www.google.com">.
Pour insérer une image <img src="w3schools.jpg" width="104" height="142">.
De plus on peut ajouter un 'alt' à l'image qui sera afficher si on ne peut la charger.

Les headings

On les utilise pour hiérarchiser le contenu.
Ils commencent par h et sont au nombre de 6.(ex: <h1>)
On peut faire une règle horizontale : <hr/>

Les paragraphes

On peut faire des paragraphes avec <p>.
Si on veut garder un texte préformaté on utilise <pre>

Les styles

On peut utilise du css dans du code html avec l'attribut style="property:value;". Exemple de propriétés : 

  - background-color: couleur;
  - color: couleur;
  - border: taille type couleur;
  - font-family
  - font-size
  - text-align (left,center,right)

Formatter son texte

<b> Defines bold text
<em> Defines emphasized text
<i> Defines italic text
<small> Defines smaller text
<strong> Defines important text
<sup> Defines erscripted text
<ins> Defines inserted text
<del> Defines deleted text
<mark> Defines marked/highlighted text

Quoter

On peut utiliser l'élément <q> pour définir une petite quotation : Alea jacta est.
Sinon on peut utiliser <blockquote cite="SOURCE"> si l'on veut ajouter la source de la quote. On obtient alors :

Alea jacta est

Pour réaliser un abbréviation on utilise <abbr title="SIGNIFICATION"> . Par exemple www.
On utilise la balise <address> pour écrire un addresse.
Pour citer le nom d'une oeuvre on utilise <cite>.

Les commentaires

On ecrit des commentaires avec <!-- COMMENTAIRE -->. On peut les ecrire sur plusieurs lignes si on ne ferme pas avec -->.
On peut aussi écrire des commentaires conditionnels pour n'éxécuter du code html qu'en présence d'un certain navigateur : <!--[if IE 9]> CODE <![endif]-->.

La couleur

On a vu précédemment, dans la partie style, des propriétés permettant de colorer notre page, voyons maintenant comment. On a plusieurs moyens de choisir la couleur, en utilisant leurs : 

  - nom
  - code rgb : rgb(255, 99, 71)
  - valeur hex : #ff6347
  - valeur hsl : hsl(9, 100%, 64%)
  - code rgba : rgba(255, 99, 71, 0.5)
  - valeur hsla : hsla(9, 100%, 64%, 0.5)
Les codes rgba et hsla prennent en compte la transparence.

Le CSS

On utilise le CSS pour gagner du temps sur la mise en page. Nous avons déjà utilié du CSS dans la propriété style d'un élément.

Le CSS en ligne

Ce CSS correspond à celui utilisé au cas par cas dans la propriété style d'un élément.

Le CSS interne

Le CSS interne est utlisé dans la sectione <head> d'une page HTML. 

<head>
<style>
body {background-color: powderblue;}
h1   {color: blue;}
p    {color: red;}
</style>
</head>

Le CSS externe

Le CSS externe est écrit dans un fichier différent en .css et est appelé dans la page html via : 

<head>
<link rel="stylesheet" href="styles.css">
</head>

Le fichier doit alors ressembler à : 

body {
    background-color: powderblue;
}
h1 {
    color: blue;
}
p {
    color: red;
}

Utilisation du CSS

Voici une liste de propriétés utilisables : 

  - background-color:
  - color:
  - font-family:
  - font-size:
  - border:
  - padding: (espace entre le texte et la bordure)
  - margin: (espace en dehors de la bordure)
  - ...

Pour modifier un élément particulier en peut utiliser un id, par exemple : <p id="p01"> Je suis différent </p>. Puis dans le code CSS on fait appel à cet id avec #p01 {}. L'id doit être unique.
Pour modifier plusieurs éléments de même type, on peut ajouter un attribut classe de la façon suivante <p class="test"> la </p> pour ensuite l'appeler via p.test {} dans le fichier CSS.

Les liens

Les liens permettent de passer à un autre document. Ils sont délimité par la balise <a href="LIEN">.
L'apparence des liens est par défaut en bleu et souligné pour un lien non visité, en violet et souligné pour un lien visité et en rouge et souligné pour un lien actif.
On peut changer la couleur des liens avec un fichiers CSS en modifiant les propriétés color, background-color, text-decoration de a:link, a:visited, a:hover et a:active.

L'attribut target permet d'ouvrir la page selon un ordre spécifié. 

    _blank : dans une nouvelle fenêtre ou un nouvel onglet
    _self : dans la même fenêtre/onglet (par défaut)
    _parent :
    _top : dans tout le corps de la fenête
    _framename : dans une fenêtre nommée

On peut utiliser une image comme lien, c'est à dire qu'on peut faire :
<a href="LIEN"><img src="" alt="" style=""></a>

On peut aussi utiliser des liens internes : des bookmarks. Pour cela, on remplace le lien dans href par un id --> <a href="#C4">.

Les images

On utilise la balise <img/> pour ajouter une image. Cette balise comprend trois attributs :

  • src: lien de l'image
  • alt: message à afficher si l'image ne peut être chargée
  • style: dimensions de l'image
Ce qui donne <img src="url" alt="some_text" style="width:width;height:height;">

On peut ajouter la propriété float:left(ou right) à style pour faire flotter une image à gauche ou à droite d'un texte.

On peut utiliser une map pour mettre plusieurs liens sur une image, en fonction de là où on a cliqué. 

    <img src="planets.gif" alt="Planets" usemap="#planetmap" style="width:145px;height:126px;">

    <map name="planetmap">
      <area shape="rect" coords="0,0,82,126" alt="Sun" href="sun.htm">
      <area shape="circle" coords="90,58,3" alt="Mercury" href="mercur.htm">
      <area shape="circle" coords="124,58,8" alt="Venus" href="venus.htm">
    </map>
